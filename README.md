# Assignment-4
Constraint Satisfaction Problems (CSP)

This repository contains Python implementations of different CSP problems using backtracking and constraint techniques.

Programs Included
1. Australia Map Coloring
Colors Australian states so that no adjacent states have the same color.
Uses backtracking.
Output:
Colored graph using matplotlib.
2. Telangana District Coloring
Assigns colors to Telangana districts.
Ensures no neighboring districts have the same color.
Output:
District → Color mapping printed in console.
3. Sudoku Solver
Solves a 9×9 Sudoku puzzle.
Uses constraint propagation and backtracking.
Output:
Displays input Sudoku grid.
Displays solved Sudoku grid using matplotlib.
4. Cryptarithmetic Puzzle Solver
Solves puzzles like:
SEND + MORE = MONEY
Each letter represents a unique digit.
Output:
Displays valid digit-letter mapping.
Shows the solved equation.
