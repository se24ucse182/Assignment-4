# Assignment-4
#  Constraint Satisfaction Problems (CSP) in Python

This repository contains implementations of different **Constraint Satisfaction Problems (CSPs)** using Python.  
Each problem is solved using techniques like **backtracking** and **constraint propagation**.

---

##  Programs Included

### 1. Australia Map Coloring
- Solves the map coloring problem for Australian states.
- Ensures no two neighboring states share the same color.
- Uses **backtracking algorithm**.
-  Output: Visual colored graph using matplotlib.
